## Design Context

### Users
External consultants and advisors who run compliance self-assessments across multiple client organizations. They spend significant time in the tool, need to quickly parse dense data, and present findings to stakeholders. Efficiency and professionalism are paramount — the tool reflects on their credibility.

### Brand Personality
**Premium, precise, refined.** Think Stripe's dashboard — meticulous attention to detail, quiet confidence, and understated elegance. Every typographic choice should feel intentional, not default. The interface should communicate competence and trustworthiness without being cold or corporate.

3-word personality: **Precise. Elevated. Confident.**

### Aesthetic Direction
- Light and dark themes (both required)
- Current accent: cyan/teal (#0891b2)
- Clean surfaces with clear hierarchy
- Data-dense but never cluttered
- Stripe, Linear, and Vercel as reference points for quality bar
- Anti-references: generic SaaS templates, overly playful UI, cluttered enterprise software

### Design Principles
1. **Typography carries the design** — In a data-heavy compliance tool, type is the primary visual element. Every size, weight, and spacing choice must be deliberate.
2. **Quiet confidence over loud decoration** — Let whitespace, hierarchy, and precision do the work. No gratuitous effects.
3. **Information density with clarity** — Consultants need to see a lot at once. Dense is fine; confusing is not.
4. **Consistency is credibility** — Inconsistent styling undermines trust. Same role = same treatment, always.
5. **Dark mode is a first-class citizen** — Not an afterthought or inversion. Both themes should feel intentionally designed.
